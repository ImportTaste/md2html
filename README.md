# Markdown to HTML cli tool

## usage

If you have Go installed, install with:

    go get -u github.com/ttys3/md2html

To run:

    mdtohtml [options] inputfile [outputfile]

or

    cat inputfile | mdtohtml [options]

Run `md2html` to see all options.
